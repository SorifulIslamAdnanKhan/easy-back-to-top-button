== Description ==
**Easy Back To Top Button** is a lightweight, user-friendly plugin that allows you to add a customizable "Back to Top" button to your WordPress website. 

No coding skills are required! Customize button appearance, including color, size, border radius, and icon size, directly from the WordPress Customizer. This tool is perfect for enhancing user experience with smooth scrolling functionality.

### Features:
- Easy to install and configure.
- Customize button background color, icon color, width, height, and border radius.
- Responsive design – works perfectly on all devices.
- Lightweight and optimized for performance.
- Smooth scroll animation powered by GSAP.
- FontAwesome-based "back-to-top" icon.

**Why choose Easy Back To Top Button?**
- Improves user navigation and accessibility.
- Enhances website usability with smooth scrolling.
- Easily configurable through WordPress Customizer.

**Perfect for:**
- Blogs, portfolios, business websites, and WooCommerce stores.

Give your visitors an easy way to scroll back to the top of the page with this simple yet powerful plugin.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/easy-back-to-top-button` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to **Appearance > Customize > Easy Back To Top Button** to configure the button.
4. Adjust the colors, sizes, and other settings to match your website design.
5. Save and publish your changes.

== Frequently Asked Questions ==
= How do I change the button color and size? =
You can customize the button color, border radius, icon size, and other settings from **Appearance > Customize > Easy Back To Top Button**.

= Will this plugin slow down my website? =
No. The plugin is lightweight, optimized for performance, and uses clean code practices.

= Does it work on mobile devices? =
Yes, the button is fully responsive and works seamlessly on all screen sizes.

= Can I disable the button on specific pages? =
Currently, the plugin doesn’t provide a per-page control, but this feature may be added in a future update.
